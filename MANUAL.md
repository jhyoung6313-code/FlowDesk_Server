## S-01 로그인

**경로:** `/login`

### 일반 로그인
1. 아이디와 비밀번호를 입력합니다.
2. **로그인** 버튼을 클릭합니다.
3. 로그인 성공 시 대시보드(`/`)로 이동합니다.
4. 로그인 실패 시 화면에 오류 메시지가 표시됩니다.

### OTP 2단계 인증 로그인 (OTP 설정 계정)
1. 아이디·비밀번호 입력 후 **로그인** 클릭.
2. OTP 입력 화면으로 전환됩니다.
3. Google Authenticator(또는 호환 앱)에서 6자리 코드를 확인하여 입력합니다.
4. OTP 인증 성공 시 대시보드로 이동합니다.
   > OTP 임시 토큰은 **5분** 이내에 인증해야 합니다.

### 신규 계정 신청 (회원가입)
1. 로그인 화면에서 **계정 신청** 링크를 클릭합니다.
2. 이름, 아이디, 비밀번호(6자 이상), 이메일을 입력합니다.
3. **신청** 버튼 클릭 → TOTP 시크릿 및 QR 코드가 발급됩니다.
4. Google Authenticator 앱에서 QR 코드를 스캔합니다.
5. 앱에 표시된 6자리 OTP를 입력하면 계정이 활성화되고 자동 로그인됩니다.

### 비밀번호 초기화
1. 로그인 화면에서 **비밀번호 초기화** 링크를 클릭합니다.
2. 아이디를 입력 → OTP 인증 화면으로 전환.
3. OTP 앱에서 코드를 입력합니다.
4. 새 비밀번호(6자 이상)를 입력하고 확인합니다.

## S-02 대시보드

**경로:** `/`

전체 업무 현황을 한눈에 파악할 수 있는 요약 화면입니다.

### 주요 구성

| 영역 | 내용 |
|------|------|
| **상태 카드** | 전체·대기·진행중·완료·보류 업무 수 표시 |
| **상태별 도넛 차트** | 업무 상태 비율을 원형 차트로 시각화 |
| **파트별 누적 바 차트** | 파트별 완료·진행중·대기 업무 수 (스택 바) |
| **담당자별 바 차트** | 담당자 기준 업무 수 (상위 8명) |
| **금일 할일** | 오늘 마감인 업무 목록 |
| **이번 주 마감** | 이번 주 내 마감 예정 업무 목록 |

### 사용 팁
- 차트 항목을 클릭하면 해당 조건으로 필터링된 업무 목록으로 이동합니다.
- 이번 주 마감 목록에서 업무명을 클릭하면 업무 상세 드로어가 열립니다.

## S-03 업무 목록

**경로:** `/tasks`

등록된 전체 업무를 목록으로 조회하고 관리하는 화면입니다.

### 필터 기능
화면 상단 필터 영역에서 다음 조건으로 업무를 필터링할 수 있습니다.

| 필터 | 설명 |
|------|------|
| 파트 | 특정 담당파트 선택 |
| 담당자 | 특정 팀원 선택 |
| 상태 | 대기 / 진행중 / 완료 / 보류 |
| 우선순위 | 높음 / 보통 / 낮음 |
| 기간 | 마감일 범위 지정 |
| 검색 | 업무명 키워드 검색 |

### 정렬 기능
- 마감일 / 우선순위 / 생성일 기준으로 오름차순·내림차순 정렬 가능

### 목록 주요 동작

| 동작 | 방법 |
|------|------|
| 업무 상세 보기 | 업무명 클릭 → 상세 드로어 열림 |
| 상태 즉시 변경 | 상태 배지(뱃지) 클릭 → 드롭다운에서 선택 |
| 업무 수정 | 수정 아이콘(✏️) 클릭 → 수정 드로어 열림 |
| 업무 삭제 | 삭제 아이콘(🗑️) 클릭 → 확인 후 삭제 |
| 캘린더로 이동 | 마감일 옆 📅 아이콘 클릭 → 해당 월 캘린더로 이동 |

### 데이터 내보내기

| 버튼 | 설명 |
|------|------|
| **Excel 내보내기** | 현재 필터 적용된 업무를 `.xlsx` 파일로 다운로드 (`업무목록_YYYY-MM-DD.xlsx`) |
| **PDF 출력** | 현재 필터 적용된 업무를 A4 가로 PDF로 다운로드 (`업무목록_YYYY-MM-DD.pdf`) |

### 업무 상태 표시
- **마감 경과:** 빨강 강조 표시
- **D-day 뱃지:**
  - `D+n` — n일 경과
  - `D-Day` — 마감 당일
  - `D-n` — n일 남음

### 업무 등록
화면 우상단 **+ 업무 추가** 버튼 클릭 → 등록 드로어 열림 (→ [S-04 참조](#s-04-업무-등록수정-드로어))

## S-04 업무 등록·수정 (드로어)

업무 목록 화면 우측에서 슬라이드로 열리는 패널입니다.

### 기본 정보 입력

| 필드 | 설명 | 필수 |
|------|------|------|
| 업무명 | 업무 제목 | ✓ |
| 설명 | 업무 상세 설명 | |
| 담당파트 | 파트 목록에서 선택 | |
| 담당자 | 팀원 선택 또는 외부인 직접 입력 | |
| 시작일 / 마감일 | 날짜 범위 선택 (각각 단독 입력 가능) | |
| 우선순위 | 높음 / 보통 / 낮음 | |
| 상태 | 대기 / 진행중 / 완료 / 보류 | |
| 의존업무 | 선행 업무 선택 (Finish-to-Start) | |
| 태그 | 다중 선택 가능 | |

### 담당자 지정 방법
- **등록 팀원:** 목록에서 선택 → 이니셜+고유 색상 아바타로 표시
- **미등록 외부인:** 이름 직접 입력 후 Enter → 회색 아바타로 표시
- 등록 팀원과 외부인을 혼합하여 복수 지정 가능

### 템플릿 활용

| 기능 | 방법 |
|------|------|
| 템플릿 불러오기 | 드로어 상단 **템플릿 불러오기** 버튼 → 목록에서 선택 → 폼 자동 입력 |
| 템플릿으로 저장 | 폼 작성 후 **템플릿으로 저장** 버튼 → 템플릿명 입력 → 저장 |

### 상세 드로어 탭 구성
업무명 클릭 시 열리는 **상세 드로어**에는 다음 탭이 포함됩니다.

| 탭 | 내용 |
|------|------|
| **상세** | 업무 기본 정보 및 수정 |
| **댓글** | 팀원 간 의견 작성 및 조회 |
| **첨부파일** | 파일 업로드·다운로드·삭제 |
| **히스토리** | 변경 이력 타임라인 |
| **시간 기록** | 타임트래킹 타이머 |

#### 댓글
- Enter 키 또는 전송 버튼으로 등록
- 본인 댓글 인라인 수정 가능
- 본인·Admin 삭제 가능

#### 첨부파일
- 드래그&드롭 또는 클릭으로 업로드 (최대 **20MB**)
- 파일명·크기·업로더·업로드 시각 표시
- 본인·Admin 삭제 가능

#### 히스토리
- 생성·수정·삭제·상태변경·파일첨부 이력을 역순으로 표시
- 작업자, 변경 필드, 이전값→변경값, 시각 확인 가능

#### 시간 기록 (타임트래킹)
- **시작** 버튼 클릭 → 실시간 경과시간 표시 (1초 단위)
- **정지** 버튼 클릭 → 메모 입력 후 저장
- 전체 누적 소요시간 합산 표시
- 동시에 하나의 타이머만 실행 가능

### 수정·삭제 권한

| 권한 | 수정 가능 | 삭제 가능 |
|------|-----------|-----------|
| Admin | ✓ | ✓ |
| 생성자 | ✓ | ✓ |
| 담당자 | ✓ (상태·실적) | ✗ |

## S-05 캘린더 뷰

**경로:** `/calendar`

업무 일정을 달력 형식으로 시각화하는 화면입니다.

### 뷰 전환
- 우상단 버튼으로 **월 / 주 / 일** 뷰 전환
- 좌우 화살표로 이전·다음 기간 이동

### 이벤트 표시
- 각 업무는 **첫 번째 등록 담당자의 고유 색상**으로 표시
- 마감 경과 업무: **빨강**으로 강조
- 완료 업무: **취소선 + 반투명** 처리
- 마일스톤: 🚩 아이콘으로 표시

### 주요 기능

| 기능 | 방법 |
|------|------|
| 업무 상세 보기 | 이벤트 클릭 → 팝업 확인 → 수정 드로어로 이동 가능 |
| 기한 변경 | 이벤트 드래그&드롭 → 시작일·마감일 자동 업데이트 |
| 메모 등록 | 빈 날짜 영역 클릭 → 메모 입력 모달 → 📝 아이콘으로 표시 |
| 메모 수정·삭제 | 📝 아이콘 클릭 → 등록자·Admin만 수정·삭제 가능 |

## S-06 간트 차트

**경로:** `/gantt`

업무의 기간 및 의존관계를 막대 차트로 시각화합니다.

### 화면 구성
- 좌측: 업무 목록 (파트명 prefix 포함)
- 우측: 기간 막대 및 의존관계 화살표

### 시간축 전환
- 우상단 버튼으로 **일 / 주 / 월** 축 전환

### 막대 색상 (상태별)
| 상태 | 색상 |
|------|------|
| 대기 | 파랑 |
| 진행중 | 초록 |
| 보류 | 주황 |
| 완료 | 빨강 (취소선) |

### 진행률 표시
| 상태 | 진행률 |
|------|--------|
| 대기 | 0% |
| 보류 | 20% |
| 진행중 | 50% |
| 완료 | 100% |

### 필터 기능
| 필터 | 설명 |
|------|------|
| 업무명 검색 | 키워드 입력 |
| 파트 | 드롭다운 선택 |
| 상태 | 드롭다운 선택 |
| 담당자 | 드롭다운 선택 |
| 필터 초기화 | 전체 필터 한 번에 초기화 |

### 주요 동작
- 막대 **드래그**로 시작일·종료일 변경 가능
- 막대 **툴팁** 확인: 업무명·파트·상태·기간·담당자

## S-07 사용자 관리 (Admin)

**경로:** `/admin/users` | Admin 전용

팀원 계정을 추가·관리하는 화면입니다.

### 사용자 추가
1. **사용자 추가** 버튼 클릭
2. 아이디, 이름, 비밀번호(6자 이상), 역할(Admin/Member), 담당파트 입력
3. **추가** 버튼 클릭

### 사용자 관리
| 기능 | 방법 |
|------|------|
| 파트 변경 | 해당 사용자 행의 파트 셀 수정 |
| 역할 변경 | Admin ↔ Member 전환 |
| 계정 삭제 | 삭제 아이콘 클릭 → 확인 후 삭제 |

### 계정 정보 표시
- 이름, 아이디, 역할, 담당파트, OTP 활성화 여부

## S-08 파트 관리 (Admin)

**경로:** `/admin/parts` | Admin 전용

담당파트(팀·조직 단위)를 설정하는 화면입니다.

### 파트 추가
1. 파트명 입력 (예: "기획팀", "개발1팀")
2. **추가** 버튼 클릭
3. 중복된 파트명은 등록되지 않습니다.

### 파트 수정·삭제
- 파트명 클릭 → 인라인 수정
- 삭제 아이콘 클릭 → 확인 후 삭제

## S-09 알림 목록

**경로:** `/notifications`

마감 관련 알림을 확인하는 화면입니다.

### 알림 발생 조건
| 유형 | 조건 |
|------|------|
| 마감 3일 전 | 마감일까지 3일 이내 |
| 마감 당일 | 마감일 당일 |
| 마감 경과 | 마감일 다음날부터 D+1↑ |

> 매일 **09:00**에 자동으로 알림이 생성됩니다.

### 알림 확인
- 헤더 🔔 아이콘의 숫자 뱃지: 미읽음 알림 수
- 🔔 클릭 → 알림 목록 화면으로 이동
- 알림 항목 클릭 → 읽음 처리 + 해당 업무 이동
- **전체 읽음** 버튼: 모든 알림 읽음 처리

## S-10 프로필 & 보안

**경로:** `/profile`

내 계정 정보를 관리하는 화면입니다.

### 접근 방법
- 헤더 우상단 아바타(이니셜) 클릭 → **프로필** 메뉴 선택

### 비밀번호 변경

## S-11 WBS 관리

**경로:** `/wbs`

프로젝트별 WBS(Work Breakdown Structure) 및 이슈사항을 관리하는 화면입니다.

### 화면 구성
- **좌측:** 프로젝트 목록 카드
- **우측:** 선택한 프로젝트의 상세 정보 + 시트 탭

### 프로젝트 관리 (Admin)
| 기능 | 방법 |
|------|------|
| 프로젝트 생성 | 좌측 **+ 새 프로젝트** 버튼 → 이름·시작일·종료일·역할 담당자 입력 |
| 프로젝트 수정 | 우측 상단 **수정** 버튼 |
| 프로젝트 삭제 | 우측 상단 **삭제** 버튼 → 확인 후 삭제 |

### 프로젝트 상단 정보
| 항목 | 설명 |
|------|------|
| 프로젝트명·기간 | 이름, 시작일·종료일 |
| 역할 담당자 | PM·PL 등 역할명+담당자명 쌍 (자유 입력) |
| 기준일자 | 진척률 계산 기준일 (기본값: 오늘) |
| 계획진척률 | 전체 WBS 항목의 계획진척률 평균 |
| 실적진척률 | 전체 WBS 항목의 실적진척률 평균 |
| 공정준수율 | 실적진척률 / 계획진척률 × 100 |

### [WBS] 탭

#### WBS 항목 구조
- 계층: **대·중·소 3단계** (최대 5단계)
- 부모 행의 진척률은 자식 기간 가중평균으로 자동 롤업

#### WBS 컬럼 설명

| 컬럼 | 설명 |
|------|------|
| 작업명 | 항목 이름 (계층 들여쓰기 표시) |
| 산출물명 | 해당 항목의 산출물 |
| 기간 | 시작일~종료일 자동 계산 |
| 시작일·종료일 | 직접 입력 |
| 계획진척률 | Admin이 목표 진척률 입력 |
| 실적진척률 | 담당자가 실제 진척률 입력 |
| 공정준수율 | MIN(실적/계획, 100%) |
| 지연 | 종료일 초과 + 실적 100% 미만 시 D+n 빨강 표시 |

#### WBS 항목 조작

| 기능 | 방법 |
|------|------|
| 항목 추가 | 행 우클릭(컨텍스트 메뉴) → 동일레벨 또는 하위항목 추가 |
| 항목 삭제 | 행 우클릭 → 삭제 |
| 순서 변경 | 행을 드래그&드롭으로 이동 |
| 인라인 편집 | 셀 클릭 후 직접 입력 |

#### WBS Excel 내보내기·가져오기
| 기능 | 방법 |
|------|------|
| Excel 내보내기 | 상단 **Excel** 버튼 → `.xlsx` 다운로드 |
| PDF 출력 | 상단 **PDF** 버튼 → A4 가로 PDF 다운로드 |
| Excel 가져오기 (Admin) | 상단 **가져오기** 버튼 → `.xlsx` 업로드 → 기존 데이터 대체 |

### [이슈사항] 탭

프로젝트 진행 중 발생한 이슈를 관리합니다.

#### 이슈 필드

| 필드 | 설명 |
|------|------|
| 번호 | 자동 부여 |
| 구분 | 파트·역할 또는 자유 텍스트 |
| 이슈내용 | 이슈 설명 |
| 발생일 | 날짜 |
| 목표해결일 | 해결 목표 날짜 |
| 진척률 | % 입력 |
| 완료예정일 | 날짜 |
| 상태 | open / in_progress / closed / hold |
| 비고 | 추가 메모 |

#### 이슈 조작
| 기능 | 방법 |
|------|------|
| 이슈 추가 | 하단 **+** 버튼 클릭 |
| 이슈 편집 | 셀 클릭 → 인라인 편집 |
| 상태 필터 | 상단 필터 드롭다운 선택 |
| Excel 내보내기 | **Excel** 버튼 클릭 |
| Excel 가져오기 (Admin) | **가져오기** 버튼 → `.xlsx` 업로드 |

> **이슈 수정 권한:** Admin은 전체, Member는 본인이 등록한 이슈만 수정 가능

## 가계부

**경로:** `/ledger`

팀 또는 개인의 수입·지출 거래 내역을 관리하는 화면입니다.

### 화면 구성

#### 헤더 영역
| 버튼 | 설명 |
|------|------|
| 연도·월 선택 | 조회할 기간 필터 |
| **카테고리** | 카테고리 관리 모달 열기 |
| **반복거래** | 반복 거래 관리 모달 열기 |
| **Excel** | 현재 연/월 거래 내역 `.xlsx` 다운로드 |
| **+ 내역 추가** | 거래 입력 모달 열기 |

#### 요약 카드 (3개)
| 카드 | 설명 |
|------|------|
| 총 수입 | 선택 월 수입 합계 (초록) |
| 총 지출 | 선택 월 지출 합계 (빨강) |
| 잔액 | 수입 - 지출 (양수: 파랑, 음수: 빨강) |

#### 차트 영역
- **좌측 (연간 바 차트):** 월별 수입·지출 막대 차트 (12개월)
- **우측 (도넛 차트):** 이번 달 지출 카테고리별 비율

#### 예산 현황 카드
- 예산이 설정된 지출 카테고리만 표시
- 진행 바 색상: 녹색(80% 미만) → 주황(80~100%) → 빨강(초과)
- 초과 시 ⚠️ 아이콘 + 초과 금액 툴팁

#### 거래 내역 테이블
| 필터 | 설명 |
|------|------|
| 전체 / 수입 / 지출 | 유형별 필터 (상단 탭) |
- 컬럼: 날짜, 유형, 카테고리(색상), 금액(+/-), 메모, 수정/삭제
- 20건/페이지 페이지네이션

### 거래 내역 추가·수정
1. **+ 내역 추가** 버튼 클릭
2. 다음 항목 입력:

| 항목 | 설명 |
|------|------|
| 유형 | 수입 / 지출 |
| 카테고리 | 유형에 맞는 카테고리 선택 |
| 금액 | 숫자 입력 |
| 날짜 | 거래 날짜 선택 |
| 메모 | 선택 입력 |

3. **저장** 클릭

### 카테고리 관리
카테고리 버튼 클릭 → 모달에서 탭 3개 제공

**1. 카테고리 목록 탭**
- 기존 카테고리 수정·삭제
- 기본 카테고리 12개 (수입 3개·지출 9개) 제공

**2. 카테고리 추가/수정 탭**
- 이름, 유형(수입·지출), 색상(컬러피커) 입력 후 저장

**3. 예산 설정 탭**
- 지출 카테고리별 월 예산 금액 입력
- **일괄 저장** 버튼으로 한 번에 저장

> 거래 내역이 있는 카테고리는 삭제할 수 없습니다.

### 반복 거래 관리
반복거래 버튼 클릭 → 모달에서 관리

**반복 거래 목록**
| 컬럼 | 설명 |
|------|------|
| 유형 | 수입·지출 |
| 카테고리 | 분류 |
| 금액 | 거래 금액 |
| 매월 N일 | 발생 일자 |
| 활성 | 토글로 켜기·끄기 |

**반복 거래 적용**
- **{월} 적용** 버튼: 활성화된 반복 거래를 현재 선택 월에 일괄 생성
- 같은 날짜에 이미 생성된 거래는 중복 생성되지 않음

**반복 거래 추가**
- 유형, 카테고리, 금액, 매월 일(1~28), 메모 입력 후 저장
