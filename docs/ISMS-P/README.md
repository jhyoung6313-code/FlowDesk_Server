# ISMS-P 대비 문서

ISMS-P(정보보호 및 개인정보보호 관리체계 인증)는 KISA가 운영하는 통합 인증제도입니다.
본 디렉토리는 task-manager(FlowDesk)의 ISMS-P 인증 준비 산출물입니다.

> ⚠️ **확인 필요**: ISMS-P **인증 의무 대상** 여부는 회사 매출·이용자 수·업권에 따라 결정됩니다(정보통신망법 시행령). 의무 대상이 아니어도 금융권은 자율 적용을 권장합니다. 대상 여부는 법무/컴플라이언스팀 확인이 필요합니다.

## 인증 체계 개요
ISMS-P = **ISMS(정보보호 80개 인증기준)** + **개인정보보호 추가 22개 기준** = 총 102개 인증기준

| 영역 | 분야 | 인증기준 수 |
|------|------|-----------|
| 1. 관리체계 수립 및 운영 | 4개 분야 | 16 |
| 2. 보호대책 요구사항 | 12개 분야 | 64 |
| 3. 개인정보 처리단계별 요구사항 | 5개 분야 | 22 |
