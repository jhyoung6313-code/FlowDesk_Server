# 영역 1. 관리체계 수립 및 운영

ISMS-P 영역1은 대부분 **관리적·조직적 조치**로, 코드보다 정책·조직 차원의 이행이 핵심입니다.

## 1.1 관리체계 기반 마련
| 통제 | 요구사항 | 현황 | 조치사항 |
|------|---------|------|---------|
| 1.1.1 경영진 참여 | 정보보호·개인정보보호 의사결정 체계 | ⬜ | CISO/CPO 지정 및 보고체계 수립 (확인 필요) |
| 1.1.2 최고책임자 지정 | CISO·CPO 공식 지정 | ⬜ | 임원급 지정 (전자금융감독규정 의무) |
| 1.1.3 조직 구성 | 정보보호 전담조직 | ⬜ | 전담 인력·역할 정의 |
| 1.1.4 범위 설정 | 인증 범위 문서화 | 🟡 | 본 시스템·인프라 범위 정의 필요 |
| 1.1.5 정책 수립 | 정보보호정책·시행문서 | 🟡 | `SECURITY_POLICY.md` 기반 사규화 |
| 1.1.6 자원 할당 | 예산·인력 | ⬜ | 연간 보안 예산 편성 |
